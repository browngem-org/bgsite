# bgsite

Fukuda Laboratory homepage — Next.js 15 + TypeScript + Tailwind CSS.

## Develop

```bash
npm install
npm run dev
```

Visit http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## Deploy

This repo is wired for one-click deploy on [Vercel](https://vercel.com). Import the GitHub repository — no env vars or build customization required.
